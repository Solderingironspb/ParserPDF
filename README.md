# ParserPDF by Solderingiron v1.0.1 win x64
## Программа предназначена для парсинга pdf файлов, а именно таблиц в них.
## Возможности:
* Парсинг таблиц из PDF файлов и передача данных в Excel таблицы
## О релизе:
* Python 3.13.1
* camelot-py 1.0.0
* colorama 0.4.6
* ghostscript 10.04.0
* java jre1.8.0_441 
* jpype1 1.5.2
* pandas 2.2.3
* pip 25.0
* pyinstaller 6.11.1
* pypdfium2  4.30.1
* pywin32 308
* questionary 2.1.0
* tabula-py 2.10.0
## Как пользоваться:
* Внутри архива есть Manual.pdf
